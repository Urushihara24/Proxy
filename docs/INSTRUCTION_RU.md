# Инструкция Для Заказчика (Windows)

## 1) Скачать приложение

Скачайте архив по ссылке релиза:

- [Proxy-Seller-Launcher-Windows-v1.0.0.zip](https://github.com/Urushihara24/Proxy/releases/download/v1.0.0/Proxy-Seller-Launcher-Windows-v1.0.0.zip)

## 2) Распаковать архив

Распакуйте `zip` в любую удобную папку, например:

- `Рабочий стол\Proxy Seller Launcher`

Важно:

- Не запускайте `.exe` прямо из архива.
- В папке рядом с `Proxy Seller Launcher.exe` должна лежать папка `_internal`.

## 3) Запустить приложение

Запустите:

- `Proxy Seller Launcher.exe`

Если появится Windows SmartScreen:

1. Нажмите `Подробнее`.
2. Нажмите `Выполнить в любом случае`.

## 4) Первый запуск и работа

1. Вставьте API-ключ Proxy-Seller.
2. Нажмите `Обновить данные`.
3. Выберите тип прокси и параметры (страна/период или тариф).
4. Нажмите:
   - `Подключить купленный` (если прокси уже есть), или
   - `Купить и подключить` (создаст новый платный заказ).

## 5) Полезные кнопки

- `Отключить прокси` - отключить системный прокси.
- `Ещё -> Показать активные` - посмотреть активные прокси.
- `Ещё -> Проверить баланс` - проверить баланс аккаунта.
- `Ещё -> Подключить последний` - быстро вернуть последний рабочий прокси.

## 6) Что делать, если не запускается

1. Убедитесь, что архив полностью распакован.
2. Запустите `Proxy Seller Launcher.exe` повторно.
3. Проверьте, не заблокировал ли запуск SmartScreen/антивирус.
4. Перезапустите ПК и попробуйте снова.

Если проблема остается, отправьте разработчику:

- скрин ошибки,
- версию Windows,
- на каком шаге ошибка возникает.
